# Zenith M&B

Pensado como un Producto Mínimo Viable, Zenith Mind & Body es una aplicación para dispositivos Android de bienestar general que busca ayudar al usuario en distintas áreas de bienestar, principalmente en lo emocional, mental y físico donde podrá medir y hacer seguimiento a nuevos retos que lo ayuden a tener una vida más organizada.
