# Diseño de interfaz de Usuario

La aplicación tendrá las siguientes pantallas:

## 1. Pantalla Splash:

Pantalla de bienvenida de la aplicación muestra el logo y eslogan de la aplicación

![Pantalla Splash](images/Diseno_Pantalla_1.png)

## 2. Pantalla configuración inicial:

Pantalla de configuración inicial, permite saber el nombre y las preferencias hacía de la aplicación del usuario

![Pantalla Splash](images/Diseno_Pantalla_2.png)

## 3. Pantalla de Inicio (Home):

Pantalla de inicio (home) de la aplicación, es el punto de partido hacía otras pantalla, muestra resumenes de actividad mediante tarjetas.

![Pantalla Splash](images/Diseno_Pantalla_3.png)

## 3.1. Pantalla Inicio (panel de configuración):

La pantalla de inicio cuenta con un botón de configuración que despliega las configuraciones desde la izquierda de la aplicación (cambiar tamaño de letra, temas, ver acerca de la aplicación).

![Pantalla Splash](images/Diseno_Pantalla_3_1.png)

## 3.2. Pantalla Inicio (notificaciones):

La Pantalla de inicio cuenta con un botón de notificaciones que despliega el panel de notificaciones desde la derecha

![Pantalla Splash](images/Diseno_Pantalla_3_2.png)

## 4. Pantalla Registro de estado de ánimo:

Pantalla de registro de estados de animo del usuario, muestra un connjutno de emociones para que el susuairo los selccione de acuerdo a sus estado de ánimo, también cuenta con un switch que le permite ver el historial de registros.

![Pantalla Splash](images/Diseno_Pantalla_4.png)

## 4.1. Pantalla Registro de estado de animo (registro):

Al seleccionar un estado de ánimo, aparece una ventana emergenete que le solicita (opcional) al usuario una descripción de cómo se siente, luego guarda dicho estado

![Pantalla Splash](images/Diseno_Pantalla_4_1.png)

## 4.2. Pantalla Registro de estado de animo (historial)

Al cambiar el switch, se muestra el historial de registro de estados de animo a través del tiempo, la gráfica muestra distinatas emociones y su tamaño de acuerdo a la frecuencia en la que se registran.

![Pantalla Splash](images/Diseno_Pantalla_4_2.png)
