# Zenith M&B

Zenith M&B es una aplicación para android de bienestar que busca ayudar al usuario en tres áreas importantes: 

**Registro de su estado de ánimo**, el usuario podrá llevar un diario de su estado de ánimo mediante la selección de algunos emojis predefinidos que refleje cómo se siente en el transcurso del tiempo.

**Funciones:**
 - Registro de estado animo mediante selección de emojis.
 - Registro opcional del motivo del por qué se siente como se siente.
 - Visualización del historial de registros mediante grafica dinámica.

**Creación de tareas o actividades diarias**, el usuario podrá crear, editar, borrar y/o programar actividades que lo ayuden a organizar sus que hacer.

**Funciones:**
- Creación, edición, eliminación y/o actualización de actividades.
- Marcar y llevar un registro de las tareas ya realizadas.
- Ver una lista de actividades que se llevaran a cabo en su día.
- Ver las tareas sin completar o próximas a realizar mediante calendarios semanales y mensuales.

**Rutinas de ejercicio**, el usuario podrá elegir algún ejercicio de una lista predefinida de actividades física y configurar series, repeticiones y tiempos de descansos para empezar una rutina deportiva.

**Funciones:**
- Selección de una actividad fisica de una lista ya defina.
- Configuración de tiempos de entrenamientos y descansos.
- Visualización y configuración de cronometro durante el tiempo de entrenamiento.
